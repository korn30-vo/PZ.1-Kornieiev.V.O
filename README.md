## Тема
Попередня обробка даних та класифікація методами машинного навчання.

## Опис
У роботі досліджено основні способи підготовки даних до навчання моделей: бінаризацію, масштабування, нормалізацію та кодування міток. Реалізовано класифікацію за допомогою Logistic Regression, Naive Bayes та Support Vector Machine.

## Використані технології
Python, NumPy, Pandas, Matplotlib, Scikit-learn.

## Результат
Побудовано моделі класифікації та виконано оцінювання їх якості за метриками Accuracy, Precision, Recall, F1-score та ROC-AUC.
